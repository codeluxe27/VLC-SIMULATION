# 🎥 VLC Player Simulation

Une simulation réaliste d'un lecteur multimédia VLC faite avec **HTML, CSS et JavaScript**.

Même si ce n'est pas une réplique à 100 %, les contrôles classiques de lecture sont fonctionnels : lecture/pause, volume, barre de progression, plein écran, etc.

## ✨ Fonctionnalités

- ▶️ Lecture / Pause
- 🔊 Contrôle du volume
- ⏱️ Barre de progression (seek)
- 🖥️ Mode plein écran
- 🎨 Interface inspirée de VLC (design sombre)
- Responsive (fonctionne sur mobile et desktop)

## 🛠 Technologies utilisées

- HTML5
- CSS3
- JavaScript (Vanilla)
